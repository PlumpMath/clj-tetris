Sample Tetris game in Clojure/Script

[demo](https://rawgit.com/yogthos/Clojure-Tetris/master/demo.html)

to compile Clojure version run
```bash
lein uberjar
```
to compile ClojureScript version run
```bash
lein cljsbuild once
```
